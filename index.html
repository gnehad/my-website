Qu'est-ce que la programmation web ?

La programmation web consiste à créer des sites ou applications accessibles via un navigateur. Elle repose sur plusieurs langages qui travaillent ensemble.

HTML

HTML (HyperText Markup Language) permet de structurer le contenu d’une page web : titres, paragraphes, images, liens, etc.

CSS

CSS (Cascading Style Sheets) permet de styliser le contenu HTML : couleurs, mise en page, polices, animations, etc.

JavaScript

JavaScript est un langage de programmation qui permet de rendre les pages interactives : menus dynamiques, formulaires, animations, etc.
